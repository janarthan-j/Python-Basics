# 📌 Exercises

## Exercise 1: Student Details

Write a Python program that does the following:

* Store the following details using variables:

  * Student name
  * Age
  * Height (in meters)
  * Whether the student is enrolled (`True/False`)

* Print all the details in a readable format.

* Create another variable called `birth_year` and calculate it using the current year and the student’s age.

* Print the birth year.

### Example Output:

```python
Student Name: John
Age: 20
Height: 1.75 m
Enrolled: True
Birth Year: 2006
```
